Техническое задание (ТЗ)
1. Модуль авторизации
При запуске приложения отображается окно авторизации с полями «Логин»,
«Пароль» (скрытый ввод) и кнопками «Вход», «Выход».
Вход выполняется только для предопределённой учётной записи: логин hr,
пароль admin.
При неверном логине или пароле выводится сообщение «Ошибка авторизации.
Проверьте логин и пароль», поля очищаются.
После успешной авторизации открывается главное окно приложения, окно
авторизации закрывается.
2. Модуль управления вакансиями
Главное окно содержит таблицу со следующими колонками:
• ID;
• Название вакансии;
• Компания;
• Зарплата (от/до);
• Требования;
• Статус (открыта/закрыта);
Над таблицей размещены кнопки: «Добавить», «Редактировать», «Удалить»,
«Обновить».
Добавление: при нажатии открывается диалоговое окно с полями:
• Название вакансии;
• Компания;
• Зарплата от (число от 0 до 1000000);
• Зарплата до (число от 0 до 1000000, не менее «Зарплата от»);
• Требования;
• Статус (выпадающий список: «Открыта», «Закрыта»);
• Кнопки: «Сохранить», «Отмена».
Редактирование: двойной клик по строке или выделение строки + кнопка
«Редактировать» открывает окно с предзаполненными полями, изменение по тем же
правилам.
Удаление: перед удалением выводится подтверждение: «Удалить вакансию
"[Название]"?», кнопки «Да», «Нет».
Валидация: при неверном вводе поле подсвечивается красной рамкой, кнопка
«Сохранить» блокируется, выводится понятное сообщение об ошибке.
3. Модуль управления соискателями
• Приложение содержит вторую вкладку (TabControl) «Соискатели» с таблицей:
• ID;
• ФИО соискателя;
• Возраст;
• Образование;
• Опыт работы (лет);
• Контакты;
Реализованы CRUD-операции для соискателей по аналогии с разделом
«Вакансии».
4. Нефункциональные требования
Приложение не должно завершаться аварийно при некорректном вводе данных.
Любое исключение перехватывается и выводится в пользовательском
сообщении, не раскрывая системную информацию.
5. Требования к интерфейсу (UI/UX)
Окна имеют минимальный размер: главное окно – 1024×600, диалоговые –
400×350.
Приложение использует единую цветовую гамму (светлая или тёмная тема на
выбор).
